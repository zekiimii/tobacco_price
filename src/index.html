导入说明：

  • 支持 Excel (.xlsx, .xls) 或 CSV 文件格式
  • 表格必须包含以下列：id, brand, product_name, wholesale_price, retail_price

导入说明：

  • 支持 Excel (.xlsx, .xls) 或 CSV 文件格式
  • 表格必须包含以下列：product_name, wholesale_price, retail_price, date
  • 重要提示：日期 date 字段请使用文本格式（如：2025-01-15），不要使用日期格式，避免导入错误
  • 根据 product_name 在产品表中查找对应的产品ID
  • 其他字段（brand, brand_id, id）将被忽略
  • 如果找不到对应的产品，将报错
